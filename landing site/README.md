# ModIntel

**ModIntel** is a robust machine learning pipeline designed to augment **ModSecurity's** web application firewall capabilities. It improves threat detection accuracy while providing real-time analytics through an intuitive visual dashboard.

## 🚀 Features

- **Intelligent Precision**: Reduces false positives using advanced ML models.
- **Real-Time Analytics**: Visualizes attacks instantly via a modern dashboard.
- **Drop-in Integration**: Seamlessly augments existing ModSecurity setups.
- **Modern UI**: A premium, glassmorphic interface designed for clarity and speed.
